Project Hierarchy

Parent Projects, Child Projects, Sub-Projects

The module helps split projects as sub-projects and link to parent project.

Key Features:

  • Splitting projects as sub-projects
  • Unlimited hierarchy level
  • User can set parent project to design project hierarchy
  • Creating unlimited subprojects inside parent project
  • User can specify percentage of subproject
  • Through the settings, you can make the field "Customer" mandatory

Module Settings:

After installing the module, administrator can enable using subprojects and make the field "Customer" mandatory.

Parent Project Form:

User can add subprojects and manage them through parent project form or in new tab using smart button.

System validates sub-projects percentage

Child Projects List

If user clicks the "Sub-Projects" button, he/she can see the list of child projects to manage them

Subproject Form

  • Project manager can select parent project
  • Customer is required (it can be changes in the module settings)
  • User can set percentage of sub-project
